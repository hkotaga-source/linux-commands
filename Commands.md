1. To check the current username of the current session : "whoami"

2. To check date and time : "date"

3. To navigate between directories : "cd [directory_path]" 
   ------> cd means change directory 

4. To list all files in a detiled , long form of lists manner in the curernt directory : "ls -la"

5. To display all available storage devices such as hard drives , ssd , flash drives and partitions : "lsblk" (list block devices)

6. To reset the terminal session : "reset"

7. To exit the terminal : "exit"

8. To list your system's physical RAM and swap memory usage in a human readable format : "free -h"
   -----> It also lists buffer/cache.

9. To view real time PID's (process ID) : "top" or "htop" 
   ---------> "htop" is color-coded and scrollable 

10. To view a static one time snap shot of the PID's : "ps aux"


11. To have ultimate access : "su" and then type your paswword.
    ------> This gives you root access 

12. To create a new directory : "mkdir" (make directory) 

13. To list  available and used disk space in human-readable formats : "df -h"
